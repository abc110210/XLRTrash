# XLRTrash - 全服垃圾桶 Spigot 插件

一个功能强大的 Minecraft Spigot 服务器全服共享垃圾桶插件。

## 功能特性

- 🗑️ **个人垃圾桶**：玩家可以将不需要的物品放入个人垃圾桶
- 🌍 **全服共享**：个人垃圾桶关闭后，物品会自动转移到全服垃圾桶
- 👥 **多人共享**：所有玩家都可以从全服垃圾桶中取出有用的物品
- 📄 **分页系统**：支持多页显示，方便浏览大量物品
- 🎨 **精美界面**：使用玻璃板装饰，视觉效果美观

## 使用方法

### 命令

| 命令 | 描述 | 权限 |
|------|------|------|
| `/trash` | 打开个人垃圾桶 | 无需权限 |
| `/globaltrash` | 打开全服垃圾桶 | 无需权限 |

### 操作说明

1. **个人垃圾桶**：
   - 使用 `/trash` 命令打开
   - 可以自由放入和取出物品
   - 关闭界面后物品自动转移到全服垃圾桶

2. **全服垃圾桶**：
   - 使用 `/globaltrash` 命令打开
   - 所有玩家共享
   - 只能取出物品，不能放入
   - 点击粘液球"下一页"翻页
   - 点击青金石"上一页"返回

## 技术信息

- **Spigot API 版本**：1.21.1-R0.1-SNAPSHOT
- **Java 版本**：21
- **Minecraft 版本**：1.21.1
- **构建工具**：Maven

## 构建

### 前置要求

- Java 21 或更高版本
- Maven 3.6.0 或更高版本

### 构建步骤

```bash
# 克隆仓库
git clone <repository-url>
cd XLRTrash

# 构建项目
mvn clean package

# 生成的 jar 文件位于 target/XLRTrash-1.0.jar
```

### GitHub Actions 自动构建

本仓库包含 GitHub Actions 工作流，每次推送代码时会自动构建插件。

## 安装

1. 下载最新版本的 `XLRTrash-1.0.jar`
2. 将文件放入服务器的 `plugins` 文件夹
3. 重启服务器或使用 `/reload` 命令
4. 开始使用 `/trash` 命令！

## 配置

插件使用默认配置，无需额外配置文件。
